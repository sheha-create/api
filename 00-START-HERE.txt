================================================================================
                      API ENDPOINT REFERENCE
================================================================================

POST /detect-voice:
  Request Headers:
    - X-API-Key: your-api-key (required)
    - Content-Type: multipart/form-data

  Request Body:
    - file: Audio file (WAV or MP3)

  Response (200):
    {
      "classification": "AI-generated" | "human",
      "confidence": 0.0-1.0,
      "explanation": "Technical reasoning",
      "timestamp": "ISO 8601",
      "processing_time_ms": number
    }

  Error Responses:
    - 400: Invalid file format or empty file
    - 401: Missing API key
    - 403: Invalid API key
    - 413: File too large (>25MB)
    - 429: Rate limit exceeded
    - 500: Server error

GET /health:
  Response (200):
    {
      "status": "healthy",
      "timestamp": "ISO 8601",
      "version": "1.0.0"
    }
